# Features

- React 19 SPA with Vercel deployment and SPA routing.
- NeoBrutalism UI design with multiple selectable themes.
- Dashboard with KPI charts (Chart.js) and pipeline status.
- Direct-to-YouTube publishing engine (7-stage chunked resumable upload from Drive).
- Google Apps Script backend integrated directly with Google Sheets.
- Client-side Google OAuth2 with token lifecycle management.
